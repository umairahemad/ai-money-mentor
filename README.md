# AI Money Mentor 

##  Overview

AI Money Mentor is an AI-powered personal finance assistant that helps users plan investments, optimize taxes, analyze portfolios, and achieve financial independence. It combines financial logic with LLM-based insights to deliver smart, real-time guidance.

---

##  Problem

95% of people lack a structured financial plan. Traditional financial advisors are expensive (₹25,000+/year) and inaccessible to most users.

---

##  Solution

AI Money Mentor provides a complete financial planning system with:

* FIRE (Financial Independence, Retire Early) planning
* Tax optimization suggestions
* Portfolio analysis
* Couple financial planning
* Life event financial advice
* AI-powered insights using LLM

---

##  Features

* 🔥 FIRE Planner – Calculates retirement target and SIP
* ⚖️ Tax Calculator – Suggests best tax regime and savings
* 📊 Portfolio Analyzer – Detects losses and inefficiencies
* 👫 Couple Planner – Optimizes joint finances
* 📅 Life Event Advisor – Smart decisions for life events
* 🏥 Money Health Score – Financial wellness tracking
* 🤖 AI Assistant – Personalized financial insights using LLM

---

##  Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: FastAPI (Python)
* AI/LLM: Ollama (LLaMA3)
* API Communication: Fetch API

---

##  Architecture

Frontend (UI) → FastAPI Backend → Financial Logic + LLM → Response → UI

---

##  Setup Instructions

### 1. Clone Repository

git clone [https://github.com/umairahemad/ai-money-mentor.git](https://github.com/umairahemad/ai-money-mentor.git)

### 2. Install Dependencies

pip install -r requirements.txt

### 3. Run Backend

uvicorn main:app --reload

### 4. Run Frontend

Open `index.html` using Live Server
